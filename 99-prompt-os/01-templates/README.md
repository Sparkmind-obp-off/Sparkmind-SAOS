# SPOS Templates

> Status: Folder contract Draft SPOS-001; template pekerjaan khusus belum dibuat.

## Tujuan

Menyimpan struktur reusable untuk artefak kerja seperti architecture, research, feature, bugfix, dan refactor.

## Aturan

- Template mendefinisikan bentuk, bukan authority atau keputusan.
- Template wajib merujuk rule dan sumber upstream kanonik.
- Token placeholder harus jelas dan tidak boleh tertinggal pada artefak final.
- Template baru dibuat hanya ketika ada use case nyata dan session yang mengizinkannya.

Session Template berada di [`../03-sessions/SESSION_TEMPLATE.md`](../03-sessions/SESSION_TEMPLATE.md) karena lifecycle dan ownership-nya dimiliki komponen Sessions.
