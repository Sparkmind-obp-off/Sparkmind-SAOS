# Knowledge

> Status: Struktur aktif; knowledge base substantif belum dikurasi.

## Tujuan

Menjadi tempat pengetahuan terverifikasi, terkurasi, dan dapat digunakan ulang lintas SAOS, Engineering, dan Products.

## Ruang Lingkup

Mencakup knowledge note, synthesis, model konseptual, dan referensi terkurasi. Tidak menyimpan transcript mentah, opini tanpa status, keputusan normatif, atau salinan research source.

## Jenis Dokumen

- knowledge note dengan sumber;
- synthesis lintas research;
- conceptual model dan domain guide;
- indeks pengetahuan serta dependency map.

## Hubungan

- Menerima evidence dari [`../research/`](../research/README.md).
- Memberi konteks kepada [`../decision-library/`](../decision-library/README.md) dan [`../playbooks/`](../playbooks/README.md).
- Pembelajaran matang dapat dikaitkan ke [`../wisdom/`](../wisdom/README.md).
- Istilah mengikuti [`../terminology/`](../terminology/README.md).

## Aturan Kualitas

Setiap knowledge note harus menyebut sumber, status verifikasi, asumsi, batas berlaku, owner, dan tanggal review.
