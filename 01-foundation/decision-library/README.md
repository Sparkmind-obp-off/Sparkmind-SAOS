# Decision Library

> Status: Struktur aktif; decision record Foundation belum ditambahkan.

## Tujuan

Menyimpan keputusan penting Foundation beserta konteks, alternatif, trade-off, owner, approval, dan konsekuensinya agar dapat ditelusuri.

## Ruang Lingkup

Mencakup keputusan arsitektur informasi, governance, pengetahuan, dan playbook Foundation. Tidak menggantikan ADR Engineering, keputusan produk, atau ketentuan fundamental Kernel.

## Jenis Dokumen

- foundation decision record;
- decision index;
- supersession map;
- exception dan review record.

## Hubungan

- Menggunakan evidence dari [`../research/`](../research/README.md) dan [`../knowledge/`](../knowledge/README.md).
- Dinilai melalui principles, values, ethics, dan governance.
- Menjadi sumber bagi [`../pattern-library/`](../pattern-library/README.md), [`../anti-pattern-library/`](../anti-pattern-library/README.md), dan [`../playbooks/`](../playbooks/README.md).

## Struktur Minimum Record

Context, status, decision, alternatives, rationale, consequences, owner, approver, date, dependency, dan review trigger.
