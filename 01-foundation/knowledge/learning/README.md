# Learning

> Status: Struktur aktif; jalur belajar substantif belum disusun.

## Tujuan

Mengubah pengetahuan terkurasi menjadi jalur belajar yang membantu manusia dan AI memahami domain secara bertahap, terukur, dan sesuai peran.

## Informasi yang Disimpan

- learning objective dan prerequisite;
- learning path, module map, exercise, dan assessment guide;
- reading list menuju artefak approved;
- feedback serta gap pembelajaran.

## Aturan Pengelolaan

- Materi harus menyatakan audience, prerequisite, outcome, owner, status, dan review trigger.
- Learning tidak menjadi sumber fakta baru; semua isi substantif menautkan knowledge atau sumber kanonik.
- Materi deprecated harus dikeluarkan dari jalur default dan diarahkan ke pengganti.
- Data penilaian personal tidak disimpan pada tahap Foundation ini.

## Hubungan

Learning menggunakan concepts, glossary, architecture, standards, best practices, decisions, dan references. [`../onboarding/`](../onboarding/README.md) memilih subset learning berdasarkan peran, sedangkan wisdom dan feedback membantu memperbaiki jalur belajar.
