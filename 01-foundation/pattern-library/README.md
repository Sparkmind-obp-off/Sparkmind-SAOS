# Pattern Library

> Status: Struktur aktif; pattern substantif belum dikurasi.

## Tujuan

Menyimpan pendekatan berulang yang telah menunjukkan hasil baik dalam konteks tertentu dan dapat digunakan ulang secara bertanggung jawab.

## Ruang Lingkup

Mencakup organizational, documentation, governance, knowledge, dan decision pattern lintas domain. Tidak menetapkan kewajiban otomatis dan tidak menyimpan template tanpa evidence penggunaan.

## Jenis Dokumen

- pattern description;
- context dan problem statement;
- forces, solution, consequence, dan limitation;
- evidence serta related patterns.

## Hubungan

- Diturunkan dari [`../decision-library/`](../decision-library/README.md), outcomes, dan [`../wisdom/`](../wisdom/README.md).
- Dibandingkan dengan [`../anti-pattern-library/`](../anti-pattern-library/README.md).
- Pattern approved dapat digunakan oleh [`../playbooks/`](../playbooks/README.md).
- Ketidakpastian dikembalikan ke [`../research/`](../research/README.md).

## Guardrail

Setiap pattern wajib menyatakan konteks, kontraindikasi, evidence, owner, status, dan trigger review.
